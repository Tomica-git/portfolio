# extract_from_filenames.py 処理概要

## 何をするプログラムか

`dummy_images/` フォルダ内の画像ファイル名を解析し、
`自動入力マクロ.xlsm`「貼り付け場所変換」シートと**同等の全7列データ**を `output.csv` に出力する。

---

## 出力 CSV の列構成（Excel シートと対応）

| 列 | ヘッダー | Excel の数式 | Python の実装 |
|----|---------|-------------|--------------|
| A | ファイル名 | （貼り付け元） | `os.listdir()` で取得 |
| B | 正式型番 | `IF(MID(A,1,5)=...)` | `MODEL_MAP` 辞書で変換 |
| C | 取得位置 | `IF(MID(A,6,2)=...)` | `LOCATION_MAP` 辞書で変換 |
| D | サンプル番号 | `MID(A,8,2)*1` | `int(filename[7:9])` |
| E | 破断方向 | `IFERROR(FIND("①"...)...)` | `str.find()` で①②③を検索 |
| F | 測定値 | `MID(A, FIND("_"),...)*1` | 正規表現グループ①で抽出 |
| G | 破断状態 | `MID(A, FIND("um"),...)`  | 正規表現グループ②で抽出 |

---

## ファイル名の構造

```
84-50TE04①_7.56um破断状態A.jpg
│    ││  ││ │    │       │
│    ││  ││ │    │       └─ .jpg（拡張子）
│    ││  ││ │    └─────────── 破断状態（um〜.jpg 間）← G列
│    ││  ││ └──────────────── 測定値（_〜um 間）    ← F列
│    ││  │└─────────────────── 破断方向（①②③）    ← E列
│    ││  └──────────────────── サンプル番号（8-9文字目）← D列
│    │└─────────────────────── 取得位置（6-7文字目）← C列
└────────────────────────────── ウェハ型番略称（1-5文字目）← B列
```

## 処理フロー

```
開始

 ▼

│         │                                │
│    マッチ？                               │
│   Yes ／  No                             │
│    ↓       ↓                             │

 │
```

## 正規表現の詳細

```python
pattern = r"_([0-9]+\.[0-9]+)um(.+?)\.jpg$"
```

| パーツ | 意味 | 例 |
|--------|------|----|
| `_` | アンダースコアを起点にする | `_` |
| `([0-9]+\.[0-9]+)` | 測定値を**グループ①**として取得 → F列 | `7.56` |
| `um` | セパレータとして読み飛ばす | `um` |
| `(.+?)` | 破断状態を**グループ②**として取得 → G列 | `破断状態A` |
| `\.jpg$` | `.jpg` で終わることを確認 | `.jpg` |

### マッチ例

```
84-50TE04①_7.56um破断状態A.jpg
            ^^^^    ^^^^^^^^^
         group(1)  group(2)
          → 7.56   → 破断状態A
```

## 変換テーブル（B列・C列）

### B列: 正式型番

| 略称（先頭5文字） | 正式型番 |
|------------------|---------|
| `80-50` | `DH260507580-50` |
| `84-12` | `CE260402384-12` |
| `84-50` | `DCH003484-50` |

### C列: 取得位置

| コード（6-7文字目） | 変換結果 |
|--------------------|---------|
| `TE` | `Top Edge` |
| `BM` | `Bottom Middle` |
| `RE` | `Right Edge` |
| `LE` | `Left Edge` |
| `Ct` | `Center` |
